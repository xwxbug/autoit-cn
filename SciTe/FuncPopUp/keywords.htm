关键字 参考列表

下面是 关键字 的详细列表.  点击各个关键字名称得到详细介绍.

 

关键字 详细信息
False / True 表示逻辑表达式的布尔值.
#comments-start 注释指定的片断.(注释块).
ContinueCase 终止Select或者Switch块中的当前case循环,并继续下一个case循环.
ContinueLoop 继续一个 While/Do/For 循环.
Default 在函数CALL中使用的预定义关键字值.
Dim / Global / Local / Const 定义一个变量,常量,或者创建一个数组
Do...Until 基于一个表达式循环.
Enum 枚举常量.
Exit 终止脚本运行.
ExitLoop 终止一个 While/Do/For 循环.
For...To...Step...Next 基于表达式循环.
For...In...Next 枚举对象集合或者数组中的元素
Func...Return...EndFunc 声明一个用户自定义函数(user-defined function),并可以放置0个或者多个可选参数,也可以设置返回一个值.
If...Then 单一条件选择语句.
If...ElseIf...Else...EndIf 条件选择语句.
#include-once 指定当前文件只能被包含一次.
#include 包含一个文件到脚本中.
#NoAutoIt3Execute 指定当前已编译脚本不能使用 /AutoIt3ExecuteLine 或者 /AutoIt3ExecuteScript 参数运行.
#NoTrayIcon 脚本开始执行时不显示托盘的图标.
#OnAutoItStartRegister 注册一个当AutoIt开始运行时调用的函数.
ReDim 调整一个已经存在的数组的大小
#RequireAdmin 指定当前脚本需要拥有完整的管理员权限才能正确运行.
Select...Case...EndSelect 条件运行语句.
Static 定义一个静态变量或者静态数组.
Switch...Case...EndSwitch 条件运行语句.
While...WEnd 基于表达式进行循环.
With...EndWith 用于减少对象类型变量的参考文本,便于快速编写.
